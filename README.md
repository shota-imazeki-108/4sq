# Foursquare - Location Matching
## これは何か
4sqコンペにて使用したコードやリンク集など。

## コンペ概要
名前や位置情報、住所、カテゴリなどの情報から同じPOI(目標物)のマッチングを行う。
point of interest(POI)についてはググった方がわかりやすい。
